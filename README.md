# HowToAbroad Assignment

## Run Locally

Clone the project

```bash
  git clone https://github.com/sasananshul/how-to-abroad-assignment.git
```

Go to the client

```bash
  cd how-to-abroad-assignment/client
```

Install dependencies

```bash
  npm install
```

Start the dev server

```bash
  npm run dev
```
